# 👗 Aura Mode - AI-Powered Wardrobe Manager

**Aura Mode** is a digital solution designed to simplify daily styling. It helps users manage their wardrobe efficiently using Artificial Intelligence to suggest the perfect outfit based on the weather and occasion.

## ✨ Key Features
* **Smart Dashboard:** Real-time weather integration to recommend suitable clothing.
* **Wardrobe Management:** Digitalize your closet and keep track of your favorite pieces.
* **AI Styling:** Personalized outfit suggestions (Powered by Claude API).
* **User Authentication:** Secure Login and Registration system (Work in progress 🛠️).

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Bootstrap), JavaScript.
* **Backend:** PHP (In development).
* **Database:** MySQL.
* **Tools:** VS Code, Git/GitHub, XAMPP.
